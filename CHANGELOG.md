# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0] - 2026-03-07

### Added

- Initial release of the CloudFront CDN Terraform module
- CloudFront distribution with S3 and ALB origin support
- Origin Access Control (OAC) for secure S3 access
- Custom cache policy with configurable TTLs
- Response headers policy with security and CORS support
- CloudFront function for viewer request handling
- Lambda@Edge function association support
- Custom error response configurations
- WAF Web ACL integration
- Route 53 DNS record management
- Access logging support
- Geo restriction support
- Basic and complete usage examples
